# pre
<Separator />
The `pre` element is used to define preformatted text. It can be used to display text that has been formatted for a specific purpose, such as code or data.

Example:
```html
<pre>
  <code>
    function myFunction() {
      console.log("Hello, world!");
    }
  </code>
</pre>
```
