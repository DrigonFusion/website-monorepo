# blockquote
<Separator />
The `blockquote` element is used to define a section that is quoted from another source. It can be used to include a short quotation or to provide a citation for a longer quotation.

Example:
```html
<blockquote>
  <p>This is a blockquote.</p>
</blockquote>
```
