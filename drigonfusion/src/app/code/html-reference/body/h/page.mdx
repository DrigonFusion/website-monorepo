# h1 - h6
<Separator />
The `h1` to `h6` elements are used to define headings of different levels. They can be used to organize the content of a document and to create a visual hierarchy.

Examples:
```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```
