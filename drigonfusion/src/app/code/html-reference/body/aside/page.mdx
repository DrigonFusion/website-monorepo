# aside
<Separator />
The `aside` element is used to define content aside from the page content. It can be used to include sidebars, pull quotes, epigraphs, or other content that is not part of the main flow of the document.

Example:
```html
<aside>
  <p>This is a sidebar.</p>
</aside>
```
