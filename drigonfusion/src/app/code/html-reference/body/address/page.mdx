# address
<Separator />
The `address` element is used to define the contact information for an entity. It can be used to provide the physical address, email address, phone number, or other contact information for a person, organization, or other entity.

Example:
```html
<address>
  <p>123 Main Street</p>
  <p>Anytown, USA</p>
  <p>Phone: 555-1234</p>
</address>
```
