# audio
<Separator />
The `audio` element is used to embed audio conent in a web page.

Example:
```html
<audio controls>
  <source src="rr.mp3" type="audio/mpeg">
  <source src="rr.ogg" type="audio/ogg">
  Your browser does not support the audio tag.
</audio>
```
