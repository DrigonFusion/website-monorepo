# figure
<Separator />
The `figure` element is used to define a self-contained graphical representation of the main content of a document. It can be used to include images, icons, or other visual elements that are related to the main content.

Example:
```html
<figure>
  <img src="image.jpg" alt="An image">
  <figcaption>This is a caption for the image.</figcaption>
</figure>
```

## figcaption
The `figcaption` element is used to define a caption for a figure. It can be used to provide a brief description of the graphical representation of the main content of a document.

Example:
```html
<figure>
  <img src="image.jpg" alt="An image">
  <figcaption>This is a caption for the image.</figcaption>
</figure>
```
