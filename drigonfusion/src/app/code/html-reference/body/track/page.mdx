# track
<Separator />
The `track` element specifies text tracks for media elements.

Example:
```html
<video width="320" height="240" controls>
  <source src="rr.mp4" type="video/mp4">
  <source src="rr.ogg" type="video/ogg">
  <track src="rrsubtitles_en.vtt" kind="subtitles" srclang="en" label="English">
  <track src="rrsubtitles_ru.vtt" kind="subtitles" srclang="ru" label="Russian">
</video>
```
