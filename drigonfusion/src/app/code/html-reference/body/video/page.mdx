# video
<Separator />
The `video` element is used to embed video conent in a web page.

Example:
```html
<video width="320" height="240" controls>
  <source src="rr.mp4" type="video/mp4">
  <source src="rr.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>
```
