# ruby
<Separator />
The `ruby` element is used to define a ruby annotation. It is typically used to indicate the ruby annotation that is being used.

Example:
```html
<ruby>Ruby<rt>ルビ</rt></ruby>
```

## rt
The `rt` element is used to define the ruby text. It is typically used to indicate the ruby text that is being used.

## rp
The `rp` element is used to define the parenthesis text. It is typically used to indicate the parenthesis text that is being used.
