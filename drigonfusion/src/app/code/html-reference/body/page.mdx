# body
<Separator />
The `body` element is used to define the content of an HTML document. It can be used to include text, images, and other elements that make up the body of the document.

Example:
```html
<body>
  <h1>Hello, world!</h1>
  <p>This is a paragraph of text.</p>
</body>
```
