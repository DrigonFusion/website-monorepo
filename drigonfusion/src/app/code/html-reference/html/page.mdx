# html
<Separator />
The `html` element is the root element of an HTML document. It contains all the other elements of the document, including the head and body.

Example:
```html
<html>
  <head>
    <title>My Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>
```
