# noscript
<Separator />
The `noscript` element is used to define fallback content for a web page that does not support JavaScript. It can be used to provide alternative content or functionality for users who do not have JavaScript enabled.

Example:
```html
<noscript>
  <p>JavaScript is required to view this page.</p>
</noscript>
```
