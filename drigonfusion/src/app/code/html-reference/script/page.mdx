# script
<Separator />
The `script` element is used to include [JavaScript](/code/javascript-reference/) code in an HTML document. It can be used to add interactivity to the document, such as event listeners and animations.

Example:
```html
<script>
  function myFunction() {
    console.log("Hello, world!");
  }
</script>
```
