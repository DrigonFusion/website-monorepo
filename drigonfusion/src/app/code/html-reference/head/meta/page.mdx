# meta
<Separator />
The `meta` element is used to specify metadata about the document. It can be used to provide information about the document, such as the title, description, and keywords.

Examples:
```HTML
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<meta name="description" content="This is a description of the document." />
<meta name="keywords" content="keyword1, keyword2, keyword3" />
```
