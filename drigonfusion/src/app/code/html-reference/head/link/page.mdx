# link
<Separator />
The `link` element is used to specify a link to an external resource. It can be used to link to stylesheets, images, and other resources.

Examples:
```html
<link rel="stylesheet" href="styles.css" />
<link rel="icon" href="favicon.ico" />
<link rel="sitemap" href="sitemap.xml" />
```
