# head
<Separator />
The `head` element is the first child of the `html` element and contains metadata about the document, such as the title of the page and links to external stylesheets and scripts.

Example:
```html
<head>
  <title>My Web Page</title>
  <meta charset="UTF-8" />
</head>
```
