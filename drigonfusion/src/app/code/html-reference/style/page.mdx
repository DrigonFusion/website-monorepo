# style
<Separator />
The `style` element is used to define style rules (also known as [CSS](/code/css-reference/)) for an HTML document. It can be used to apply styles to elements, such as fonts, colors, and layout.

Example:
```html
<style>
  element {
    color: red;
  }
  .class {
    background-color: blue;
  }
  #id {
    font-size: 20px;
  }
</style>
```
